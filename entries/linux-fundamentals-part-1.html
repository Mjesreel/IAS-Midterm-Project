Linux Fundamentals Part 1

Navigating the command line—the foundation of both attack and defense.

Linux

What I Learned

Linux is everywhere in cybersecurity. Most servers run Linux, many security tools are built for Linux, and understanding Linux is essential for both attackers and defenders. This was my introduction to working with Linux through the command line interface (CLI).

The Command Line

At first, the command line felt intimidating. No buttons to click, no visual interface—just a black screen and a blinking cursor. But once I got comfortable, I realized how powerful it is.

I learned basic commands like:

  • ls - list files and directories
  • cd - change directory
  • pwd - print working directory (where am I?)
  • cat - view file contents
  • grep - search for text in files
  • chmod - change file permissions

Permissions and Users

One of the most important concepts I learned was the Linux permission system. Every file and directory has permissions that control who can read, write, or execute it.

Permissions are divided into three groups:

  • Owner: The user who owns the file
  • Group: A group of users
  • Others: Everyone else

Understanding permissions is crucial because many security vulnerabilities come from misconfigured permissions. If a sensitive file is readable by everyone, that's a problem.

Services and Processes

I also learned about services (programs that run in the background) and how to manage them. Commands like systemctl let you start, stop, and check the status of services.

This is important because attackers often target running services to find vulnerabilities. As a defender, you need to know what services are running and make sure they're properly secured.

Why This Matters

Linux fundamentals are the foundation of cybersecurity work. Whether you're analyzing logs, running security tools, or exploiting vulnerabilities, you'll be doing it on Linux. Getting comfortable with the command line opens up a whole world of possibilities.

"The command line is not just a tool—it's a way of thinking. It teaches you to be precise, efficient, and to understand what's really happening under the hood."
